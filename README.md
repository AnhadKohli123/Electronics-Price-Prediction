# Electronics Price Prediction using Machine Learning

This project builds a machine learning model that predicts the price of electronic products using product features.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Model Used
Random Forest Regressor

## Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Project Structure
electronics_price_prediction.ipynb → main notebook containing the model and analysis

## How to Run
1. Open the notebook
2. Run all cells
3. The model trains and predicts electronics prices
