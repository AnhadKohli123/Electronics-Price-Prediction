
# Electronics Price Prediction

This project builds a machine learning model to predict the price of electronic products using product features.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Model Used
Random Forest Regressor

## Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Project Structure

electronics_price_prediction.ipynb → main notebook  
README.md → project explanation  

## How to Run

1. Open the notebook
2. Run all cells
3. The model will train and display predictions
